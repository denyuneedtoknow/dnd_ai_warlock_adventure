# 🎲 Діоніс — D&D Campaign Tracker

Сайт для відстеження кампанії D&D 5e.

## Розділи
- **Персонаж** — характеристики, скіли, риси
- **Інвентар** — paperdoll + рюкзак + гаманець
- **Закляття** — Pact Magic слоти, cantrips, відомі закляття, Invocations
- **Персонажі** — NPC з фільтром по ставленню та локації
- **Щоденник** — хроніка сесій

## Оновлення даних
Відкрий `/admin.html`, введи GitHub PAT один раз. Далі — все через форми в браузері.

## Деплой
GitHub Pages → Settings → Pages → Source: `main` branch, `/ (root)`.
